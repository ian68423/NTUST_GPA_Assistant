# 台科大 GPA 查詢助手 (NTUST GPA Assistant) 隱私權政策

本隱私權政策旨在說明「台科大 GPA 查詢助手」（以下簡稱本套件）如何處理、儲存及保護使用者的資料。本套件致力於確保您的隱私安全，並嚴格遵守相關開發者規範。

## 1. 資料收集說明
本套件為達成單一功能目標（整合顯示學術 GPA 數據），僅會存取並處理實現該功能所必須之資料：

* **驗證資訊**：本套件會讀取使用者於設定介面輸入之電子郵件地址（帳號）與密碼。
* [cite_start]**網站內容**：本套件會從 `myntust.com` 網頁中擷取特定學術數據，包括課程代碼（例如 ADG019301） [cite: 32, 33][cite_start]、授課教師姓名（例如 吳榮梅） [cite: 51]、以及對應的平均 GPA 成績（例如 3.74）。

## 2. 資料使用用途
* **自動化登入**：所收集之驗證資訊（帳號與密碼）僅用於執行 `myntust.com` 平台的自動化登入流程。
* **數據顯示**：擷取之網站內容僅用於將 GPA 數據回顯至「國立臺灣科技大學課程查詢系統」之列表介面中，協助使用者進行選課參考。

## 3. 資料儲存與安全性
* **本地端儲存**：本套件採用的所有資料處理均發生在您的個人裝置上。帳號、密碼及 GPA 快取數據皆僅儲存於瀏覽器的本地空間（`chrome.storage.local`）。
* **無遠端傳輸**：本套件**不具備**任何遠端伺服器通訊能力。我們絕不會將您的個人資訊或擷取之數據傳送、分享或出售給任何開發者或第三方機構。

## 4. 單一用途聲明 (Single Purpose Statement)
根據 Chrome 線上應用程式商店規範，本套件僅具有範圍有限且清楚易懂的單一用途：協助使用者於台科大選課系統中直接瀏覽歷史 GPA 數據。本套件不包含任何廣告追蹤、非相關領域的行為分析或額外的數據採集。

## 5. 使用者權限
* **資料控制**：使用者可隨時透過小工具設定介面修改帳密資訊。
* **資料刪除**：若您解除安裝本擴充功能，所有儲存在 `chrome.storage.local` 中的本地資料將會隨之完全移除。

## 6. 聯繫我們
若對本政策有任何疑問，請透過 Chrome 線上應用程式商店中註冊的聯絡電子郵件與開發者取得聯繫。
