# NTUST_GPA_Assistant

# 台科大 GPA 查詢助手 (NTUST GPA Assistant) 🎓

這是一款專為國立臺灣科技大學學生設計的 Chrome 擴充功能。它可以自動整合「選課系統」與「myNTUST GPA 查詢系統」，讓你在選課時直接在列表看到該課程的歷史 GPA 紀錄，無需手動跳轉。

## ✨ 特色功能
- **自動登入**：預先設定 myNTUST 帳密，一鍵完成自動化登入。
- **背景查詢**：開啟「快速查詢模式」，點擊後在背景靜默擷取資料，不干擾目前瀏覽。
- **GPA 快取**：查詢過的 GPA 會紀錄在瀏覽器中，下次瀏覽同門課時自動顯示。
- **無資訊提示**：若該課尚無 GPA 資料，會自動標記，避免重複查詢。
- **快速返回**：在 GPA 頁面提供懸浮關閉按鈕，秒速回到選課清單。

## 🛠️ 安裝教學 (開發者模式)
由於本套件尚未上架至 Chrome 線上商店，請依照以下步驟手動安裝：

1. 點擊本頁面右上角的 `Code` -> `Download ZIP` 並解壓縮。
2. 開啟 Chrome 瀏覽器，進入擴充功能管理頁面 `chrome://extensions/`。
3. 開啟右上角的 **「開發者模式」** 開關。
4. 點擊左上角 **「載入解壓縮擴充功能」**，選擇剛才解壓縮的資料夾。
5. 在工具列將「台科大 GPA 查詢助手」釘選，方便隨時設定。

## 📝 使用說明
1. **設定帳密**：點擊小工具圖示，輸入您的 myNTUST 電子郵件與密碼。
2. **切換模式**：
   - 關閉「快速查詢」：會跳轉至 myNTUST 讓你查看詳細圓餅圖與評價。
   - 開啟「快速查詢」：分頁在後台處理，資料擷取完會自動消失，焦點留在選課網。
3. **人工認證**：若短時間查詢過快導致出現 Cloudflare 驗證，請暫時關閉快速查詢，手動完成一次驗證即可恢復。

## 🔒 隱私與安全
- 本套件僅要求 `ntust.edu.tw` 與 `myntust.com` 相關網域權限。
- 您的帳號密碼僅儲存在您本機電腦的 `chrome.storage.local` 空間，**絕對不會**傳送到任何外部伺服器。
- 建議不要在公共電腦上使用此套件。
